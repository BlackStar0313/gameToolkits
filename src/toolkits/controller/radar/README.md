# 属性雷达图
![GitHub set up](https://github.com/BlackStar0313/gameToolkits/blob/master/imgPreview/ctrRadar.png?raw=true)
### 描述
- 矢量图显示
- 图片纹理显示(暂未添加)
- 属性数组对应多边形顶点位置

	`基数多边形，原点最上方点为第一个属性，其他属性顺序为顺时针。`
	`偶数多边形，原点最右边点为第一个属性，其他属性顺序为顺时针。`

### 目前支持
- 矢量图的绘制，
- 根据传入点信息数组长度觉得正多边形边数;
- 修改正多边形外接圆半径大小;
- 修改属性名称文本，颜色;
- 修改属性值文本，颜色，显示百分比还是数值;
- 修改背景线条颜色，粗细;
- 修改属性范围的线条大小，颜色;
- 修改属性范围的背景颜色;
- 修改属性对应点的显示与否，大小，颜色;
- 支持创建绘制，延迟绘制刷新
